# Charza Instituto de Wushu

Sitio web oficial de **Charza Instituto de Wushu** en Ciudad Juárez, Chihuahua.

## Publicación Recomendada

Repositorio:

```text
CHARZA-INSTITUTO-DE-WUSHU
```

GitHub Pages URL esperada:

```text
https://aarondpana.github.io/CHARZA-INSTITUTO-DE-WUSHU/
```

## Contenido

- `index.html`: página web standalone con estilos, logo, contacto, horarios, logros y recursos.
- `.nojekyll`: archivo para que GitHub Pages sirva el sitio sin procesamiento Jekyll.
- `.github/workflows/pages.yml`: flujo de GitHub Actions para desplegar el sitio en GitHub Pages.
